# 项目基线管理系统 — 功能说明

> **版本**: v1.1.0 | **更新日期**: 2026-07-22

---

## 一、系统概述

项目基线管理系统是基于 Aras PLM 平台的项目计划基线管理工具，采用 **Vue 3 + Element Plus + Vite + TypeScript** 技术栈构建。

### 核心能力

| 能力 | 说明 |
|------|------|
| 快照管理 | 冻结项目计划状态，创建、查看、删除快照 |
| 基线比对 | 左右树状并排对比两个快照，字段粒度差异检测 |
| 关键路径分析 | 基于 CPM 算法，追踪关键路径变化 |
| AI 智能分析 | Agnes AI 流式分析，多维度解读差异 |
| 报告导出 | Excel 表格 / Word 文档一键导出 |

---

## 二、功能模块

### 1. 快照管理

**入口**：导航栏 → 快照管理

- 查看所有已冻结的项目计划快照
- 每个快照包含：名称、描述、创建时间、总任务数、完成百分比、关键路径任务数
- 支持删除快照（点击删除按钮）

**快照结构**：
```
Snapshot
├── 元数据 (名称/描述/创建时间/项目编号)
├── 统计信息 (总任务/已完成/完成百分比)
├── 关键路径 (关键任务ID列表)
└── 任务树 (WBS递归嵌套)
```

### 2. 基线比对

**入口**：导航栏 → 基线比对

#### 2.1 快照选择

- 左侧选择 **基准 A**（旧基线）
- 右侧选择 **对比 B**（新基线）
- 支持交换按钮快速切换
- 快照列表表格支持点击选号和单选按钮两种方式
- 点击 **"开始比对"** 按钮进入比对结果页

#### 2.2 比对结果页

结果页包含以下区域：

**差异统计概览**
- 新增/删除/变更/无变化 任务数量
- 彩色卡片可视化展示

**任务计划左右比对树**
- 左侧 = 基准A，右侧 = 对比B，逐行对齐
- 颜色标注：🟢 新增 / 🟠 变更 / 🔴 删除
- 显示每个任务的：WBS、名称、序号、前置依赖、状态、工期、进度条、日期范围、负责人
- 变更字段红框（旧值）/ 橙框（新值）高亮

**变更任务字段详情**
- 仅显示有变更的任务
- 每个任务一张卡片，表格展示字段/旧值/新值
- 涵盖 18 种字段类型的对比

**关键路径差异分析**
- 新增到关键路径 / 从关键路径移除的任务列表
- 总浮时变化 / 关键路径工期变化数值

### 3. AI 智能分析

**位置**：比对结果页 → AI 智能分析卡片

#### 3.1 启动分析

- 点击 **"开始 AI 分析"** 按钮
- 系统自动将差异数据构建为结构化 prompt
- 调用 Agnes AI (`agnes-2.0-flash`) 流式接口

#### 3.2 分析维度

| 维度 | 内容 |
|------|------|
| 📋 变更概述 | 总结本次基线变更的规模和影响范围 |
| 🛤️ 关键路径影响 | 分析关键路径工期变化对项目交付的影响 |
| ⚠️ 风险识别 | 识别进度、依赖、合规、资源等风险 |
| 👥 资源变化分析 | 人员变更影响、单点瓶颈识别 |
| 🏗️ 结构变化解读 | WBS结构变化背后的业务调整思路 |
| 📌 建议与结论 | 可操作的管理建议和总体结论 |

#### 3.3 交互体验

- **流式输出**：AI 逐字返回分析内容，实时滚动到最新位置
- **光标闪烁**：分析进行中显示紫色闪烁光标动画
- **状态标签**：分析中 / 已完成 / 出错
- **重新分析**：完成后可点击刷新重试
- **错误重试**：失败时显示错误原因 + 重试按钮

### 4. 报告导出

#### 4.1 导出入口

- **页头下拉菜单**："导出报告" → 选择导出类型
- **AI 卡片底部**：基线比对建议导出按钮（AI分析完成后显示）

#### 4.2 导出类型

| 导出项 | 格式 | 文件名 | 内容概述 |
|--------|------|--------|----------|
| 任务计划左右比对 | Excel (.xlsx) | `任务计划左右比对_日期.xlsx` | 完整任务比对表 + 差异统计Sheet |
| 变更任务字段详情 | Word (.docx) | `变更任务字段详情_日期.docx` | 差异摘要 + 逐任务变更明细表格 |
| 基线比对建议 | Word (.docx) | `基线比对建议_日期.docx` | AI分析报告（标题/加粗/列表/表格） |

## 三、技术架构

### 目录结构

```
src/
├── types/index.ts          # 核心类型定义
├── store/baseline.ts       # Pinia 状态管理
├── router/index.ts         # Vue Router 路由
├── utils/
│   ├── diffEngine.ts       # 差异比对引擎
│   └── criticalPath.ts     # 关键路径法(CPM)算法
├── services/
│   ├── aiAnalysis.ts       # AI 流式分析服务
│   └── exportService.ts    # 导出服务(Excel/Word)
├── mock/mockData.ts        # 模拟数据
├── views/
│   ├── HomeView.vue        # 首页
│   ├── SnapshotListView.vue # 快照管理
│   ├── CompareView.vue     # 比对选择页
│   └── CompareResultView.vue # 比对结果页
└── components/
    ├── CompareTreeView.vue # 左右树状比对组件
    └── TaskTreeDiff.vue    # 任务树差异组件
```

### 数据流

```
快照数据 (mockData.ts / Aras API)
    │
    ▼
Pinia Store (baseline.ts)
    │
    ├── selectSnapshots() → runComparison()
    │       │
    │       ▼
    │   diffEngine.compareSnapshots()
    │       │
    │       ├── buildCompareTree()  → CompareRow[]
    │       ├── buildSummary()      → DiffSummary
    │       └── compareCriticalPath() → CriticalPathDiff
    │
    ├── AI分析 → streamAIAnalysis() → 流式输出
    │
    └── 导出 → exportService.ts → Excel/Word 下载
```

## 四、操作流程

### 典型工作流

```
1. 打开系统首页 → 查看最近快照
          │
2. 进入快照管理 → 查看/删除快照
          │
3. 进入基线比对 → 选择基准A和对比B
          │
4. 点击"开始比对" → 查看比对结果
          │
5. 点击"开始AI分析" → 查看AI分析报告
          │
6. 点击"导出报告" → 选择导出格式 → 下载文件
```

| 快照 V1.1 - 设计完成 | 2026-06-28 | 设计阶段完成时状态冻结 |
