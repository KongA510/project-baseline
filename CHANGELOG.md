# 更新日志

本文档记录项目基线管理系统的所有重要变更。

---

## [v1.0.0] - 2026-07-21

### 新增
- 🎉 **项目初始化**：Vue3 + Element Plus + Vite + TypeScript 技术栈搭建
- 📸 **快照管理**：冻结项目计划状态，支持创建、查看、删除快照
- 🌲 **左右树状比对**：两个完整项目计划树并排显示，逐行对齐
- 🎨 **颜色标注差异**：
  - 🟢 绿色 = 新增任务
  - 🟠 橙色 = 变更任务
  - 🔴 红色 = 删除任务
- 📊 **关键路径法(CPM)**：基于 Aras `Critical_Path_Method.htm` 文档实现
  - 正向遍历 (Forward Pass)：ES → EF
  - 逆向遍历 (Backward Pass)：LF → LS
  - 拓扑排序处理依赖关系
  - 总浮时/自由浮时计算
- 🗂️ **模拟数据**：智能装配线开发项目（4阶段、18个任务）
  - 3个预置快照（基线 V1.0、快照 V1.1、快照 V2.0）
  - V1→V2 模拟真实变更：新增/删除/工期/状态变化
- 🤖 **AI分析区域**：预留流式分析UI框架，待接入API
- 📋 **变更详情表**：字段级别前后值对比（旧值/新值）
- 🔗 **前置依赖显示**：支持 FS/SS/FF/SF 四种依赖类型
- 📍 **排序序号N显示**：每个任务显示 `N:{sortOrder}`
