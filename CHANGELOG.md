# 更新日志

本文档记录项目基线管理系统的所有重要变更。

## [v1.1.0] - 2026-07-22

### 新增
- 🤖 **AI 流式智能分析**：集成 Agnes AI (`agnes-2.0-flash`)，对基线差异进行多维度分析
  - 流式逐字输出，实时滚动，光标闪烁动画
  - 分析维度：变更概述、关键路径影响、风险识别、资源变化、结构解读、建议与结论
  - 支持重新分析，错误重试
- 📥 **三合一导出功能**：
  - **任务计划左右比对** → Excel 表格（含差异统计 Sheet）
  - **变更任务字段详情** → Word 文档（表格式逐字段新旧值对比）
  - **基线比对建议** → Word 文档（AI 分析结果转格式文档）
- 🎨 **导出下拉菜单**：页头"导出报告"按钮下拉选择导出类型
- ⚙️ **Vite 代理配置**：转发 `/api/agnes` 到 Agnes AI 接口

### 变更
- AI 智能分析区域从"待接入 API"占位替换为真实交互组件
- 基线比对建议导出入口移至 AI 分析卡片底部

## [v1.0.0] - 2026-07-21

### 新增
- 🎉 **项目初始化**：Vue3 + Element Plus + Vite + TypeScript 技术栈搭建
- 📸 **快照管理**：冻结项目计划状态，支持创建、查看、删除快照
- 🌲 **左右树状比对**：两个完整项目计划树并排显示，逐行对齐
- 🎨 **颜色标注差异**：
  - 🟢 绿色 = 新增任务
  - 🟠 橙色 = 变更任务
  - 🔴 红色 = 删除任务
- 📊 **关键路径法(CPM)**：基于 Aras `Critical_Path_Method.htm` 文档实现
  - 正向遍历 (Forward Pass)：ES → EF
  - 逆向遍历 (Backward Pass)：LF → LS
  - 拓扑排序处理依赖关系
  - 总浮时/自由浮时计算
- 🗂️ **模拟数据**：智能装配线开发项目（4阶段、18个任务）
  - 3个预置快照（基线 V1.0、快照 V1.1、快照 V2.0）
  - V1→V2 模拟真实变更：新增/删除/工期/状态变化
- 🤖 **AI分析区域**：预留流式分析UI框架，待接入API
- 📋 **变更详情表**：字段级别前后值对比（旧值/新值）
- 🔗 **前置依赖显示**：支持 FS/SS/FF/SF 四种依赖类型
- 📍 **排序序号N显示**：每个任务显示 `N:{sortOrder}`
